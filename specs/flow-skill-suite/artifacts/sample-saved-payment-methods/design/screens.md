# Screens

## Checkout Payment Step

Data shown:

- Saved card brand
- Last four digits
- Expiration date

Actions:

- Select saved method
- Use a new payment method
- Continue checkout

States:

- No saved methods
- Saved methods available
- Saved method declined
- Saved method expired

## Account Payment Methods

Data shown:

- Saved card brand
- Last four digits
- Expiration date

Actions:

- Remove saved method

States:

- Empty account
- One or more saved methods
- Remove confirmation
