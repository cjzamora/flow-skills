# Add Saved Method Selection to Checkout

Status: Ready
Assignee:
External link:
Last updated: 2026-07-02

## What to Build

Let returning customers select a saved payment method during checkout and recover with a new payment method when the saved one fails.

## Acceptance Criteria

- [ ] Returning customers see saved payment methods on the payment step.
- [ ] Saved methods show brand, last four digits, and expiration date.
- [ ] Customers can choose a different payment method.
- [ ] Failed saved methods show a clear error and recovery path.

## Blocked By

None - can start immediately.
